# MOSAIC AAAI 2027 v5 Release

Release date: 2026-07-19

Immutable release tag: `mosaic-aaai2027-v5`

Use `mosaic_aaai2027_anonymous.pdf` for double-blind review. The v4 tag is an
older manuscript and must not be used for review of the final revision.

## Main changes since v4

- Recast the contribution around stochastic release under structured deployment
  shift, rather than validation-only candidate certification.
- Added the bridge theorem, finite-transform guarantee, capacity fallback, and
  identifiability/impossibility results with full supplement proofs.
- Added the direct target-calibrated comparator and California-to-Texas ACSIncome
  confirmation study.
- Expanded and sharpened the relationship to Learn Then Test, conformal risk
  control, distributionally robust optimization, domain adaptation, and
  randomized classifiers.
- Reworked the abstract, introduction, results, figures, limitations, and prose
  after adversarial review.
- Rebuilt anonymous and named main papers and supplements from the same source.
